Netflix Data Analysis using Python 
Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The analysis helps in understanding Netflix content trends, ratings, genres, release years, country-wise distributions, and overall content patterns through data visualization techniques.

The project was developed using Jupyter Notebook and includes data cleaning, preprocessing, statistical analysis, and graphical representations to extract meaningful insights from the dataset.

Key Insights
Movies are more dominant than TV Shows on Netflix.
The United States contributes the highest number of titles.
Netflix content significantly increased after 2015.
Drama and International Movies are among the most popular genres.
TV-MA is one of the most frequently used ratings.
