# Netflix Data Analysis using Python 

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The analysis helps in understanding Netflix content trends, ratings, genres, release years, and country-wise distributions through various data visualization techniques.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Missing Values Handling
- Movies vs TV Shows Analysis
- Ratings Distribution
- Country-wise Content Analysis
- Release Year Trend Analysis
- Genre Analysis
- Director Analysis
- Data Visualization

## Visualizations
- Bar Charts
- Pie Charts
- Histograms
- Heatmaps
- Line Plots

## Key Insights
- Movies are more dominant than TV Shows on Netflix.
- The United States has the highest amount of Netflix content.
- Content production increased rapidly after 2015.
- Drama and International Movies are among the most popular genres.
- TV-MA is one of the most common ratings.

## Learning Outcomes
This project improved practical skills in:
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization
- Python Programming
- Business Insight Generation

## Conclusion
The project demonstrates how Python can be used to analyze and visualize real-world datasets effectively. It also showcases beginner-level data analytics skills suitable for portfolio and resume projects.
